# assi_bot (Assistant telegram bot)

## Requirements
- Python 3 or later

## Setting
```
$ git clone https://github.com/byung-u/assi_bot
$ cd assi_bot
$ pip3 install -r requirements.txt
$ python3 assi_bot/assi_bot.py
```
